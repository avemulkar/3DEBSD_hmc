Misorientatio and Quant Compare

compareMisQuantN(MisOBound, min, max)
    compareMisQuantN compares a specific set of misorientation information
    to a series boundary files min to max. If using the same misorientation
    from a previous run, the informatino is stored in vals.txt.
    This produces visualization of how the quantization and misorientation
    boundaries compare as well as the the number unmatched and matched misorientatino
    and quantization points. See the m file for more information.
